# Next.js Template

A Next.js 16 template with React 19, Tailwind CSS v4, TypeScript, and Bun.

## Getting Started

```bash
bun install
bun dev
```

## Commands

| Command | Purpose |
|---------|---------|
| `bun dev` | Start development server |
| `bun build` | Build for production |
| `bun start` | Start production server |
| `bun lint` | Run ESLint |
| `bun typecheck` | Run TypeScript type checking |

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS v4
- TypeScript
- Bun
